# Bundled level packs

- `CCLP1.dat` — Chip's Challenge Level Pack 1, created by the Chip's
  Challenge community (Bit Busters Club, https://bitbusters.club).
  Community level packs are freely distributable. Downloaded from the
  Gliderbot set archive.

The original 149 Chip's Challenge levels are NOT bundled — they are
copyrighted. The game can load a user-supplied CHIPS.DAT instead.
